EVENTOS DO DOM NO JAVASCRIPT

Tipos de eventos mais importante no JavaScript

Eventos de Mouse

  • click => Ocorre quando um elemento e clicado
  • mouseover => Ocorre quando o ponteiro do mouse entre em contato com um elemento
  • mouseout => ocorre quando o ponteiro do mouse sai de um elemento.
  • mousemove: ocorre quando o ponteiro do mouse é movido dentro de um elemento.

Eventos de Teclado

  • keydown => Ocorre quando uma tecla é pressionada.
  • keyup => Ocorre quando uma tecla é liberada.
  • keypress => Ocorre quando uma tecla é pressionada e liberada.

Eventos de Formulario

  • submit => Ocorre quando um formulário é submetido.
  • focus => Ocorre quando um elemento de formulário recebe o foco.
  • blur => Ocorre quando um elemento de formulário perde o foco.
  • change => Ocorre quando o valor de um elemento de formulário é alterado.
  • input => Ocorre quando o valor de um elemento de entrada é alterado.

Eventos de Janela

  • load => Ocorre quando a página é totalmente carregada.
  • resize => Ocorre quando a janela do navegador é redimensionada.
  • scroll => Ocorre quando o conteúdo da página é rolado.

Eventos de Tempo

  • setTimeout => Ocorre após um intervalo de tempo especificado.
  • setInterval => Ocorre em intervalos de tempo regulares.

Para adicionar e remover um evento de um elemento

  • addEventListener => Usada para adicionar o evento no elemento HTML.
  • removeEventListener => Usada para remover o evento no elemento HTML.

Localizando elementos html

  • document.getElementById => Localizar elementos pelo Id
  • document.getElementsByTagName => Localizar elementos pela Tag
  • document.getElementsByClassName => Localizar elementos pela classe do elemento
  • document.querySelectorAll => Localizar elementos atraves de todos os seletor do css
  • document.querySelector => Localizar elementos atraves de seletores do css

Alterando estilo do HTML

  • document.style.color => Altera a cor da fonte do elemento.
  • document.style.backgroundColor => Altera a cor de fundo do elemento
  • document.style.fontSize => Altera o tamanho da fonte do projeto.
  • document.style.marginLeft => Altera o tamanho da margem esquerda.
  • document.style.marginRight => Altera o tamanho da margem direita
  • document.style.padding => Altera o padding de todos os lados do elemento.

Obs: Ao manipular o estilo pelo DOM do JavaScript, usamos as mesmas propriedade que usamos quando manipulamos o estilo pelo Css.
